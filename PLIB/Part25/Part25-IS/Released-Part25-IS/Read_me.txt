This directory contains the lattest versions of the released EXPRESS schemas of ISO IS 13584-25 (.exp) and of example physical files if errorsb have been discovered since publication.

 These corrected versions supersede the published versions of ISO IS 13584-25. Their version number is greater that 1.0. They are prepared by the ISO 13584-25 team project leader to solve the errors identified in the published version. 

Technical corrigenda will gather all these corrections. Before this publication, released schemas with the corresponding release notes are provided here for convenience
