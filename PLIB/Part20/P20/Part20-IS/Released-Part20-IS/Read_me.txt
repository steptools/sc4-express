This directory contains the latest versions of the released EXPRESS schemas of ISO IS 13584-20:1998 (.exp) if errors have been discovered in these schemas or files since publication.

 These corrected versions supersede the published versions of ISO IS 13584-20. Their version number is greater that 1.0. They are prepared by the ISO 13584-20 team project leader to solve the errors identified in the published version. 

Technical corrigenda will gather all these corrections. Before such publication, released schemas with the corresponding release notes are provided here for convenience
